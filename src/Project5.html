Project Overview

React Meals is an online food catering company where convenience meets culinary delight! The platform offers a seamless experience for customers to browse through a diverse selection of meals, add and/or remove them to their carts, and easily finalize their purchases. the interface ensures a hassle-free ordering process, allowing customers to effortlessly navigate through meals, select their desired items, and complete their transactions with ease.

Design brief

The brief was to create a React application that fetches external data and handling cart functionalities solely through JavaScript. The goal was to implement features to dynamically display fetched data and manage cart operations like adding and removing items using React components..
